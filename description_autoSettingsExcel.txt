Для определения настроек для загрузки экселевского файла был написан класс autoSettingsExcel.
Он имеет следующие атрибуты:
1. book - книга с данными
2. sheet_names - список названий листов книги
3. sheets_count - число листов в книге
4. isXls - переменная типа Bool, для корректного чтения файлов с расширением .xls
(если True, то читаемый файл имеет расширение .xls)

Для опредления настроек необходимо выполнить следующие действия:
1. settings = autoSettingsExcel() - создать объект класса
2. settings.set_book(filename) - установить путь к файлу
3. st = settings.get_auto_settingsForSheet(settings.get_sheet_names()[1])
- получить рекомендуемые настройки для определенного листа в книге, передав её название или номер
4. settings.check_settings(filename, settings.get_sheet_names()[1], st) 
- так же можно проверить правильность полученных настроек, передав методу путь к файлу, название листа и проверяемые настройки

Основной метод в классе - get_auto_settingsForSheet(), которому передается название или номер листа в книге,
 работает по следующему алгоритму:
1. используя название или номер листа читаются данные в нем и сохранются в виде DataFrame
2. удаляются пустые строки и столбцы, и столбцы, в которых только одна непустая ячейка
    с помощью метода self.del_blank_rows_and_columns()
3. определяются номера первой и последней строк заголовка с помощью метода __header_rows(), 
    которому передается DataFrame с данными без пустых строк и столбцов:
    3.1 определяется первая строка заголовка, как строка с наибольшим количеством непустых ячеек
    (число непустых ячеек в сроке запоминается)
    3.2 ищется последняя строка заголовка, до тех пор, пока количество непустых ячеек не меняется или строка не будет числовой
4. определяются номера первой и последней строк значащих данных с помощью метода __meaning_rows(),
    которому передается DataFrame с данными без пустых строк и столбцов, и без строк заголовка:
    4.1 удаляются столбцы с одной непустой ячейкой
    4.2 определяется первая строка значащих данных, как первая числовая строка
    4.3 определяется последняя строка значащих данных, как первая числовая строка с конца
5. определяются номера столбцов с данными с помощью метода __meaning_columns(),
    которому передается DataFrame с строками только значащих данных,
    как номера столбцов, в которых больше одной непустой ячейки
6. возвращается словарь с рекомендуемыми настройками загрузки
